# GeometryShapesSketcher
a wpf control can be used to sketch common shapes like rectangle, ellipse. it is extensible to support more shapes.
本项目主要实现如何在wpf下绘制各种图形，基于drawingVisual比wpf自带图形控件性能上更具优势。
